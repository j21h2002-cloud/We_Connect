# 카카오 알림톡 발송 시스템 — 업무 인수인계 문서

> 작성일: 2026-03-04
> 시스템명: 카카오 알림톡 발송 관리 시스템
> 문서 목적: 담당 부서 인계 및 운영 셋업 안내

---

## 1. 시스템 개요

사내 PC 어디서든 브라우저로 접속하여 카카오 알림톡을 발송하는 웹 시스템입니다.

| 항목 | 내용 |
|------|------|
| 접속 방식 | 웹 브라우저 (Chrome 권장) |
| 인터넷 연결 | 필요 (외부 호스팅) |
| 설치 필요 | 없음 (URL 접속만으로 사용) |
| 비용 | Supabase 무료 / Netlify 무료 / 알림톡 발송 건당 약 8~15원 |

### 주요 기능
- **고객사 관리**: 회사명·담당자·연락처·산업군·비고 등록/수정/삭제, Excel 일괄 가져오기
- **메시지(템플릿) 관리**: 카카오 승인 템플릿 등록, 미리보기
- **알림톡 발송**: 고객사 검색·선택 → 메시지 선택 → 발송
- **발송 이력**: 발송 일시·발송자·템플릿·성공/실패 건수 기록
- **사용자 관리**: 계정 추가/삭제, 권한(관리자/일반사용자) 설정, Excel 일괄 등록

## 3. 서비스 오픈을 위해 담당 부서가 해야 할 일

아래 3단계를 순서대로 진행하면 실제 서비스가 가능합니다.

---

### STEP 1. Supabase 설정 (데이터베이스)
**소요 시간: 약 30분**
**담당: IT 또는 시스템 담당자**

1. [https://supabase.com](https://supabase.com) 접속 → 회사 이메일로 가입
2. 새 프로젝트 생성 → 지역: **Northeast Asia (Seoul)**
3. 프로젝트 생성 후 **Settings > API** 메뉴에서 아래 2가지 값 복사:
   - `Project URL` (예: `https://abcdefgh.supabase.co`)
   - `anon public key` (긴 문자열)
4. **SQL Editor** 메뉴 → `supabase/schema.sql` 파일 내용 전체 붙여넣기 → 실행
5. **Authentication > Settings**에서 **"Confirm email"** 옵션 **OFF** (이메일 인증 비활성화)
6. `js/config.js` 파일 상단의 아래 두 줄 수정:
   ```javascript
   const SUPABASE_URL    = 'https://abcdefgh.supabase.co'   // ← 실제 URL로 변경
   const SUPABASE_ANON_KEY = 'eyJhbGci...'                  // ← 실제 키로 변경
   ```

---

### STEP 2. Netlify 배포 (웹 호스팅)
**소요 시간: 약 20분**
**담당: IT 또는 시스템 담당자**

1. [https://netlify.com](https://netlify.com) 접속 → GitHub 계정으로 가입
2. GitHub에 `카카오_알림톡_웹` 폴더를 새 저장소로 업로드
3. Netlify에서 **"Add new site > Import an existing project"** → GitHub 저장소 선택
4. 배포 완료 후 **Site settings > Environment variables** 에서 환경변수 추가:
   ```
   없음 (현재 코드에서는 별도 환경변수 불필요)
   ```
5. 배포된 URL(예: `https://kakao-alimtalk.netlify.app`) 로 접속 확인

> 💡 **사내 전용으로 사용하려면**: Netlify의 Password Protection 기능 또는 사내 서버에 직접 배포 가능 (IT 담당자 협의)

---

### STEP 3. 알리고(알림톡 발송사) 가입 및 채널 개설
**소요 시간: 2~4주 (카카오 승인 기간)**
**담당: 마케팅 또는 운영 담당자**

#### 3-1. 알리고 가입
1. [https://www.aligo.in](https://www.aligo.in) → 회원가입 (사업자 가입 권장)
2. 서비스 신청 → **카카오 알림톡** 선택

#### 3-2. 카카오 비즈니스 채널 개설 (약 2~4주 소요)
1. [https://business.kakao.com](https://business.kakao.com) → 카카오 비즈니스 계정 생성
2. **카카오톡 채널** 개설 (회사 공식 채널)
3. 채널 개설 후 알리고에 **발신 프로필(Sender Key)** 등록 요청
4. 카카오 심사 완료 후 발신 가능 상태 확인

#### 3-3. 메시지 템플릿 등록 (카카오 심사: 약 3~5 영업일)
- 발송할 알림톡 메시지 내용을 카카오 양식에 맞게 작성하여 심사 신청
- 심사 승인 후 **템플릿 코드** 발급 → 시스템의 "메시지 관리" 화면에 등록

#### 3-4. API 키 시스템에 등록
시스템 접속 후 **⚙️ 설정 메뉴**에서:
- `API Key` (알리고 App Key)
- `Sender Key` (발신 프로필 키)
- `알리고 아이디`
- `발신 번호`
입력 후 저장

---

### STEP 4. 최초 운영 계정 설정
**소요 시간: 약 10분**
**담당: 시스템 관리자**

1. Supabase **SQL Editor**에서 최초 관리자 계정 생성:
   ```sql
   -- 1) 직접 Auth 사용자 생성은 Supabase 대시보드 Authentication > Users > Invite user
   -- 또는 시스템 접속 후 설정 > 사용자 관리 > 추가 버튼 사용
   ```
2. 시스템 접속 → **⚙️ 설정 > 사용자 관리**에서 직원 계정 일괄 등록:
   - `사용자_등록_양식.xlsx` 다운로드 버튼으로 양식 받기
   - 양식에 아이디·성명·비밀번호·권한 입력 후 **일괄 등록** 버튼으로 업로드

## 5. 고객사 데이터 입력 방법

**고객사 관리 페이지**에서 Excel 일괄 등록 가능:

1. **고객사 관리** 메뉴 → **Excel 양식 다운로드** 버튼 클릭
2. 양식(`customers_template.xlsx`) 에 데이터 입력:

| 컬럼 | 설명 | 필수 |
|------|------|------|
| 회사명 | 고객사 이름 | ✅ |
| 담당자 | 수신자 성명 | - |
| 연락처 | 핸드폰 번호 (010-XXXX-XXXX) | 발송 필수 |
| 산업군 | 업종 분류 (드롭다운 선택) | - |
| 비고 | 메모 사항 | - |

3. **Excel 가져오기** 버튼으로 업로드

### 자주 발생하는 상황

| 상황 | 조치 방법 |
|------|-----------|
| 로그인이 안 됨 | 설정 > 사용자 관리에서 계정 확인 또는 비밀번호 재설정 |
| 발송이 안 됨 | 설정 메뉴에서 API Key·Sender Key 입력 확인 |
| 고객사가 안 보임 | 연락처(핸드폰 번호) 미등록 고객사는 발송 불가 |
| 템플릿 코드 오류 | 카카오에서 승인된 템플릿 코드인지 확인 |
